Uniswap Task

How to access

      http://localhost:3000/
      +
      /tokens?sortBy=tradeVolumeUSD&limit=100&page=2

      Where:
      "sortby" can have these values: 
      “tradeVolumeUSD”,
      “totalLiquidity”,
      “untrackedVolumeUSD”.

      Value of limit can also be changed

      For example:
      
      http://localhost:3000/tokens?sortBy=tradeVolumeUSD&limit=50
